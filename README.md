# Proyectos Freelance Para Tesis

[English](#freelance-thesis-projects-english) | [Español](#proyectos-freelance-para-tesis)

## Descripción

Este repositorio contiene una colección de 16 proyectos freelance realizados para tesis de estudiantes. Estos proyectos fueron desarrollados siguiendo los requisitos del cliente, incluyendo documentación, pruebas, clases y proporcionando tutoría sobre programación a los estudiantes para la defensa de la tesis. Los proyectos abarcan diferentes tipos de aplicaciones, como páginas web, servidores, bots de Telegram, aplicaciones móviles y programas de escritorio.

## Proyectos

A continuación, se enumeran algunos de los proyectos destacados realizados:

1. **bot-telegram-AMRA**: Bot de telegram para proporcionar información a estudiantes y representantes sobre actividades, este proyecto quedó funcionando como parte de la institución educativa debido a su buen funcionamiento y rendimiento.

2. **Nutriperfec**: Aplicación móvil dirigido a personas con trastornos alimenticios, esta aplicación ofrece opciones para dietas, calendario de seguimiento, notificaciones, recordatorio para tomar agua, control de peso(IMC) chatbot para preguntas sobre los trastornos alimenticios, tips, artículos, registro de estado de animo.

3. **PuzzleSpectro**: Es un programa de escritorio para Windows dirigido a niños con autismo. Este programa ofrece una colección de puzzles con niveles diseñados específicamente para brindar entretenimiento y estimulación cognitiva.

4. **PuzzleSpectro**: Es un programa de escritorio para Windows dirigido a niños con autismo. Este programa ofrece una colección de puzzles con niveles diseñados específicamente para brindar entretenimiento y estimulación cognitiva.

4. **Turboland Language Website**: Programa de escritorio desarrollado con tauri.app y react, backend nodejs, express y bd PostgreSQL, va dirigida a estudiantes de la materia idiomas, donde los estudiantes mediante su correo pueden ingresar y el profesor puede crear clases de diferentes idioma y pasar a los estudiantes: Temas, Evaluaciones, Tareas, Videos, Juegos, Libros PDF, Traductor, Audio, Notas y Asistencia.

5. **Todo Marketing**: Aplicación nativa desarrollada con react native/expo, backend con nodejs, express y bd PostgreSQL, la aplicación es una red social dirigida a emprendedores y personas que quieren encontrar emprendimientos, herramientas que destacan son ubicación de emprendimientos cercanos, interacción con publicaciones (me gusta, favoritos, guardados, comentarios, etc), chatbot con IA para (motivar a crear un emprendimiento o mejorar al emprendedor), herramientas para emprendedores que permiten crear publicaciones atractivas mediante IA y plantillas seleccionadas, esto incluye eliminación de fondo con IA para productos, generación de fondos con IA para productos, galería de imágenes y videos y perfil personalizable

6. **ScanAssist**: Aplicación nativa desarrollada con react native/expo, backend con nodejs, express y bd mySQL, que tiene el objetivo de ser usada por profesores para el registro y el control de asistencia de los estudiantes, genera automáticamente carnets para los estudiantes, permite escanear mediante QR la asistencia dependiendo de la clase y tiene sistema de envío de correo a representante en caso de ausencia a alguna clase.

7. **bot-telegram-helpTesis**: Bot creado para ayudar en el proceso de creación de tesis a los estudiantes mediante IA y prompts personalizadas, el bot funciona con gemini 1.0 pro.

8. **Inversiones Belén**:  Catálogo web desarrollado con NextJS, boundless-commerce y bd PostgreSQL para productos de un local de repuestos automotrices y zapatería, deploy en vercel.

9. **Desafío Virtual Millonario**: Software desarrollado con solo HTML, CSS y JS, simulando el icónico programa 'Quien quiere ser millonario', con preguntas básicas de computación, contiene temporizador, puntaje, pista, 50:50 y dato curioso.

10. **Bot Tattoo(Wise Body/Piel de Papel)**: Bot desarrollado para proporcionar ideas visuales hechas con inteligencia artificial que puedan inspirar diseños para tatuajes, los modelos de IA son utilizados de Hugging Face.

11. **Software-educativo**: Programa de escritorio desarrollado con tauri.app y react, está dirigido a niños con autismo y proporciona 4 actividades, rompecabezas, sopa de letras, ¿que palabra es?, ¿cual número es?

12. **AppSheet - Candy Stock**: Aplicación desarrollada con AppSheet y bd Google Sheet, esta aplicación va dirigida al control y gestión de inventario de una pastelería, esta app también permite el monitoreo mediante estadísticas.

13. **AppSheet - Insermolca**: Aplicación desarrollada con AppSheet y bd Google Sheet, esta aplicación funciona como gestor de inventario de un taller mecánico, asi como el manejo de clientes, cotizaciones, sistema para compartir la ubicación de una forma mas fácil,etc.

14. **AppSheet - Comercializadora Franklin F.P**: Aplicación desarrollada con AppSheet y bd Google Sheet, para la gestión de inventario de una tienda, este cuenta con un sistema de conversión monetaria de Bolivares(VE) a Dolares(USD) mediante una api, la conversión se realiza en todas las direcciones al crear el producto y se mantiene diariamente actualizado.

15. **AppSheet - APP-INV**: Aplicación desarrollada con AppSheet y bd Google Sheet, esta aplicación va dirigida al comedor de una escuela en la que se maneja el inventario de las comidas, también posee un sistema de notificaciones para informar a los usuarios cuando los alimentos están listos o han sido entregados

# Freelance Thesis Projects (English)

[English](#freelance-thesis-projects-english) | [Español](#proyectos-freelance-para-tesis)

## Description

This repository contains a collection of 16 freelance projects done for student theses. These projects were developed following the client's requirements, including documentation, testing, lectures and providing programming tutoring to the students for the thesis defense. The projects cover different types of applications, such as websites, servers, Telegram bots, mobile applications and desktop programs.

## Projects

Below are some of the notable projects completed:

1. **bot-telegram-AMRA**: Telegram bot providing information to students and representatives about activities. This project remained operational as part of the educational institution due to its good performance.

2. **Nutriperfec**: Mobile app for people with eating disorders, offering diet options, tracking calendar, notifications, water reminders, weight control (BMI), chatbot for questions about eating disorders, tips, articles, mood tracking.

3. **PuzzleSpectro**: Windows desktop program for children with autism, offering a puzzle collection with levels designed for entertainment and cognitive stimulation.

4. **PuzzleSpectro**: Es un programa de escritorio para Windows dirigido a niños con autismo. Este programa ofrece una colección de puzzles con niveles diseñados específicamente para brindar entretenimiento y estimulación cognitiva.

4. **Turboland Language Website**: Desktop program developed with tauri.app and react, backend nodejs, express, and PostgreSQL database, aimed at language students. Students can log in with their email, and teachers can create classes in different languages and share Topics, Evaluations, Assignments, Videos, Games, PDF Books, Translator, Audio, Notes, and Attendance.

5. **Todo Marketing**: Native app developed with react native/expo, backend with nodejs, express, and PostgreSQL database, a social network for entrepreneurs and those seeking ventures. Key features include locating nearby ventures, interacting with posts (likes, favorites, saves, comments, etc.), AI chatbot for motivation or improvement, tools for entrepreneurs to create attractive posts using AI and selected templates, including AI background removal for products, AI background generation for products, image and video gallery, and customizable profile.

6. **ScanAssist**: Native app developed with react native/expo, backend with nodejs, express, and mySQL database, aimed at teachers for student attendance registration and control. It automatically generates student ID cards, allows QR scanning for class attendance, and has an email notification system for absences.

7. **bot-telegram-helpTesis**: Bot created to assist students in thesis creation using AI and personalized prompts, functioning with gemini 1.0 pro.

8. **Inversiones Belén**: Web catalog developed with NextJS, boundless-commerce, and PostgreSQL database for automotive spare parts and shoe store products. Deployed on vercel.

9. **Desafío Virtual Millonario**: Software developed with only HTML, CSS, and JS, simulating the iconic 'Who Wants to Be a Millionaire' show with basic computing questions, featuring a timer, score, hint, 50:50, and fun fact.

10. **Bot Tattoo(Wise Body/Piel de Papel)**: Bot developed to provide visual ideas made with artificial intelligence that can inspire designs for tattoos, AI models are used from Hugging Face.

11. **Educational-software**: Desktop program developed with tauri.app and react, aimed at children with autism, providing 4 activities: puzzles, word search, which word is it?, which number is it?",

12. **AppSheet - Candy Stock**: App developed with AppSheet and Google Sheet database for a bakery inventory control and management, also allows monitoring through statistics.

13. **AppSheet - Insermolca**: App developed with AppSheet and Google Sheet database functioning as an inventory manager for a mechanical workshop, managing clients, quotes, and providing an easier way to share location, etc.

14. **AppSheet - Comercializadora Franklin F.P**: App developed with AppSheet and Google Sheet database for store inventory management, includes a currency conversion system from Bolivares (VE) to Dollars (USD) via an API, with conversion for all directions when creating a product and daily updates.

15. **AppSheet - APP-INV**: App developed with AppSheet and Google Sheet database for a school cafeteria inventory, also includes a notification system to inform users when meals are ready or delivered.
